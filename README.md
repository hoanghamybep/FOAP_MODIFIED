# foap_project

Dataset: https://drive.google.com/drive/folders/1AVXluxyY5ducIRx2l1h4seZB-it-ID0Q?usp=sharing

Please add files in traffic.zip, log.zip, and web_sample.zip to "FOAP_data/traffic/", "FOAP_data/log/", and "FOAP_data/web_sample/" respectively before running python scripts in "FOAP/".

Files in web_sample.zip are used for the metric learning that trains a mapping function for flow pattern mining.
